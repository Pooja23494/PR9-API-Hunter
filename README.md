# PR9-Api-Hunter

This is a simple API-based project built using React (Vite) and JSON Server. It allows users to fetch, create, update, and delete data using APIs.

## 🚀 Features

- Fetch data from API
- Add new data
- Update existing data
- Delete data
- Toast notifications for user feedback

--

## 🛠️ Tech Stack

- React (Vite)
- Axios API
- JSON Server
- React Toastify

--

## 📂 Project Structure

src/
 ├── assets/
 ├── App.jsx
 ├── main.jsx

## ⚙️ Installation & Setup

- 1️⃣ Install Dependencies
npm install
- 2️⃣ Run JSON Server
npx json-server --watch db.json --port 3000
- 3️⃣ Start React App
npm run dev
- 🌐 API Endpoint
http://localhost:3000/user
